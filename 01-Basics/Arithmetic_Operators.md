# ➕ Arithmetic Operators in Python

## 🧠 What are Arithmetic Operators?
Arithmetic operators are used to perform **mathematical calculations** between values.

👉 These are the most basic and most used operators in programming.

---

## 🎯 Why are they important?
- Used in every program (from basic apps to ML models)
- Help in calculations, logic building, and data processing
- Foundation for advanced topics like algorithms & data science

---

# 📌 Types of Arithmetic Operators

## 1. ➕ Addition (+)

```python

print(10 + 3)  # Output: 13

```

## 2. ➖ Subtraction (-)

```python

print(10 - 3)  # Output: 7

```

## 3. ✖️ Multiplication (*)

```python

print(10 * 10)  # Output: 100

```

## 4. ➗ Division (/)

```python

print(10 / 3)  # Output: 3.333...

```

## 5. ⬇️ Floor Division (//)

```python

print(10 // 3)  # Output: 3

```

>⚠️ Important :-  Removes decimal part and returns integer value.

## 6. 🧮 Modulus (%)

```python

print(10 % 3)  # Output: 1

```

>⚠️ Important :-  Returns the remainder after division.


## 7. 🔥 Exponent (**)

```python

print(10 ** 5)  # Output: 100000

```

> ⚠️ Important :- Means “power of a number” (10⁵).
